# ScribePad

ScribePad is a basic notes android app built using Jetpack Compose and Kotlin

## Concepts
- Jetpack Compose
- MVVM
- Clean Architecture
- Room Database
- Pluto Debugger
- Dagger/Hilt Dependency Injection

## Features
- Create and Edit Notes
- Delete Notes
- View Notes
- Color code notes
- Offline Access
